# 02-3. helloworld(name) 함수 구현

### 전체 코드 (macOS/MySQL)

```cpp
#include <mysql.h>
#include <string.h>
#include <stdio.h>
#include <stdlib.h>

#ifdef __cplusplus
extern "C" {
#endif

bool helloworld_init(UDF_INIT *initid, UDF_ARGS *args, char *message)
{
    if (args->arg_count != 1 || args->arg_type[0] != STRING_RESULT)
    {
        strcpy(message, "helloworld() requires exactly one string argument (name)");
        return 1;
    }

    initid->maybe_null = 1;
    initid->const_item = 0;
    initid->max_length = 300;
    initid->ptr = NULL;

    return 0;
}

char *helloworld(UDF_INIT *initid,
                 UDF_ARGS *args,
                 char *result,
                 unsigned long *length,
                 char *is_null,
                 char *error)
{
    if (args->args[0] == NULL)
    {
        *is_null = 1;
        return NULL;
    }

    const char *name = args->args[0];
    unsigned long name_len = args->lengths[0];

    // "Welcome " (8) + name + ", HelloWorld Mysql Plugin" (25) + null (1)
    unsigned long buf_len = 8 + name_len + 25 + 1;

    if (initid->ptr)
        free(initid->ptr);

    initid->ptr = (char *)malloc(buf_len);
    if (!initid->ptr)
    {
        *error = 1;
        return NULL;
    }

    *length = (unsigned long)snprintf(initid->ptr, buf_len,
                                      "Welcome %.*s, HelloWorld Mysql Plugin",
                                      (int)name_len, name);

    return initid->ptr;
}

void helloworld_deinit(UDF_INIT *initid)
{
    if (initid->ptr)
    {
        free(initid->ptr);
        initid->ptr = NULL;
    }
}

#ifdef __cplusplus
}
#endif
```

### 코드 상세 분석

#### `_init()`: 인자 검증

```cpp
if (args->arg_count != 1 || args->arg_type[0] != STRING_RESULT)
```

`_init()`에서 인자를 검증하는 것이 핵심입니다. 인자 검증을 `_init()`에서 하면, 쿼리가 시작될 때 한 번만 검사합니다. 반면 메인 함수에서 하면 수백만 행마다 반복 검사해야 합니다.

> **타입 강제 변환**: `args->arg_type[i]`에 원하는 타입을 직접 대입하면 MySQL이 자동으로 변환해줍니다.
> ```cpp
> args->arg_type[0] = STRING_RESULT; // 정수가 오면 문자열로 변환
> ```

#### 메인 함수: 버퍼 크기 계산

```cpp
unsigned long buf_len = 8 + name_len + 25 + 1;
```

- `8` = `"Welcome "` (공백 포함 8자)
- `name_len` = `args->lengths[0]` (바이트 단위 길이, UTF-8 멀티바이트 가능)
- `25` = `", HelloWorld Mysql Plugin"` (25자)
- `1` = null terminator

#### `snprintf`로 안전한 문자열 조합

```cpp
*length = (unsigned long)snprintf(initid->ptr, buf_len,
                                  "Welcome %.*s, HelloWorld Mysql Plugin",
                                  (int)name_len, name);
```

`%.*s`는 정밀도를 동적으로 지정하는 형식입니다. `*` 자리에 정수(`name_len`)를 전달하면 정확히 그 길이만큼만 출력합니다. 이는 null terminate되지 않은 문자열을 안전하게 처리하는 관용적인 방법입니다.

#### `_deinit()`: 메모리 해제

`_deinit()`은 쿼리가 완전히 끝난 후 딱 한 번 호출됩니다. `NULL` 체크 후 해제하고 다시 `NULL`로 초기화하는 것은 이중 해제(double free) 버그를 방지하는 방어적 코딩입니다.
