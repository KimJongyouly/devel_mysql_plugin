# 05-4. 핵심 메서드 구현

### create() — 사이드카 파일 생성

```cpp
int ha_parquet::create(const char *name, TABLE *form,
                       HA_CREATE_INFO *create_info)
{
    const char *parquet_path = nullptr;
    if (create_info->comment.str && create_info->comment.length > 0)
        parquet_path = create_info->comment.str;

    // 사이드카 파일 경로: <MySQL 데이터 경로>.prq
    char prq_path[FN_REFLEN];
    snprintf(prq_path, sizeof(prq_path), "%s.prq", name);

    FILE *f = fopen(prq_path, "w");
    if (!f) return HA_ERR_GENERIC;
    if (parquet_path)
        fputs(parquet_path, f);
    fclose(f);
    return 0;
}
```

`name`은 MySQL 데이터 디렉터리 내 테이블 경로입니다 (예: `/var/lib/mysql/mydb/sales`). 사이드카 파일은 이 경로에 `.prq` 확장자를 붙인 위치에 생성됩니다. MySQL 서버가 재시작되어도 parquet 파일 경로를 기억합니다.

### open() — Parquet 파일 로드

```cpp
int ha_parquet::open(const char *name, int mode, uint test_if_locked)
{
    share = get_share();
    if (!share) return HA_ERR_OUT_OF_MEM;
    thr_lock_data_init(&share->lock, &lock, NULL);

    char parquet_path[FN_REFLEN];
    if (!read_prq_meta(name, parquet_path))
        return HA_ERR_NO_SUCH_TABLE;

    auto file_result = arrow::io::ReadableFile::Open(parquet_path);
    if (!file_result.ok()) return HA_ERR_NO_SUCH_TABLE;

    std::unique_ptr<parquet::arrow::FileReader> reader;
    parquet::arrow::OpenFile(*file_result, arrow::default_memory_pool(), &reader);
    reader->ReadTable(&arrow_table);
    total_rows = arrow_table->num_rows();

    return 0;
}
```

`open()`은 `SELECT`가 시작될 때마다 호출됩니다. 매번 파일 전체를 메모리에 로드합니다. 대용량 파일에는 주의가 필요합니다.

### rnd_init() / rnd_next() / rnd_end() — 풀스캔

```cpp
int ha_parquet::rnd_init(bool scan) {
    current_row = 0;
    return 0;
}

int ha_parquet::rnd_next(uchar *buf) {
    if (current_row >= total_rows)
        return HA_ERR_END_OF_FILE;

    int rc = fill_row(buf, current_row);
    if (rc == 0) current_row++;
    return rc;
}

int ha_parquet::rnd_end() {
    return 0;
}
```

### fill_row() — Arrow → MySQL 타입 변환

```cpp
int ha_parquet::fill_row(uchar *buf, int64_t row_idx)
{
    memset(buf, 0, table->s->null_bytes);

    for (uint i = 0; i < table->s->fields; i++) {
        Field *field = table->field[i];
        auto column = arrow_table->column(i);

        auto chunk_result = column->GetScalar(row_idx);
        if (!chunk_result.ok()) continue;
        auto scalar = *chunk_result;

        if (!scalar->is_valid) { field->set_null(); continue; }
        field->set_notnull();

        switch (column->type()->id()) {
        case arrow::Type::INT32:
            field->store(std::static_pointer_cast<arrow::Int32Scalar>(scalar)->value, false);
            break;
        case arrow::Type::DOUBLE:
            field->store(std::static_pointer_cast<arrow::DoubleScalar>(scalar)->value);
            break;
        case arrow::Type::STRING: {
            auto s = std::static_pointer_cast<arrow::StringScalar>(scalar);
            field->store(s->value->data(), s->value->size(), &my_charset_utf8mb4_general_ci);
            break;
        }
        case arrow::Type::DATE32: {
            MYSQL_TIME mt{};
            days_to_mysql_date(
                std::static_pointer_cast<arrow::Date32Scalar>(scalar)->value, &mt);
            field->store_time(&mt);
            break;
        }
        }
    }
    return 0;
}
```

#### Field::store() 메서드

| Arrow 타입 | Field::store() 호출 |
|-----------|---------------------|
| INT32/INT64 | `store(longlong, bool unsigned_flag)` |
| FLOAT/DOUBLE | `store(double)` |
| STRING | `store(const char*, ulong len, charset)` |
| DATE32 | `store_time(MYSQL_TIME*)` |

컬럼 순서(인덱스)로 매핑합니다. parquet 파일의 0번 컬럼이 MySQL DDL의 0번 컬럼에 대응됩니다.

### days_to_mysql_date() — 날짜 변환

Apache Arrow의 `DATE32` 타입은 Unix epoch(1970-01-01) 기준 일수(정수)입니다. Howard Hinnant의 Gregorian calendar 알고리즘으로 연/월/일을 계산합니다.

```cpp
static void ha_parquet::days_to_mysql_date(int32_t days, MYSQL_TIME *out)
{
    // civil_from_days (Howard Hinnant)
    int64_t z = (int64_t)days + 719468;
    int64_t era = (z >= 0 ? z : z - 146096) / 146097;
    int64_t doe = z - era * 146097;
    int64_t yoe = (doe - doe/1460 + doe/36524 - doe/146096) / 365;
    int64_t y   = yoe + era * 400;
    int64_t doy = doe - (365*yoe + yoe/4 - yoe/100);
    int64_t mp  = (5*doy + 2) / 153;
    int64_t d   = doy - (153*mp + 2)/5 + 1;
    int64_t m   = mp < 10 ? mp+3 : mp-9;
    if (m <= 2) y += 1;

    memset(out, 0, sizeof(*out));
    out->year      = (uint)y;
    out->month     = (uint)m;
    out->day       = (uint)d;
    out->time_type = MYSQL_TIMESTAMP_DATE;
}
```
