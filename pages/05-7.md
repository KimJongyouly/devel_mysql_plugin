# 05-7. 5장 정리

- 스토리지 엔진은 `handler` 추상 클래스를 상속받아 스토리지 계층 인터페이스를 구현한다.
- `Parquet_share`는 테이블 하나당 공유되는 락과 메타정보를 보관한다.
- `open()`에서 Arrow C++ 라이브러리로 Parquet 파일을 전체 메모리에 로드한다.
- `rnd_next()`에서 커서를 한 행씩 진행하며 `fill_row()`로 Arrow 컬럼 값을 MySQL Field에 저장한다.
- 사이드카 `.prq` 파일에 parquet 경로를 저장해 서버 재시작 후에도 경로를 유지한다.
- MySQL 소스 버전과 설치된 서버 버전이 반드시 일치해야 한다.
- `build.sh`로 전체 서버를 컴파일하지 않고 필요한 부분만 선택적으로 빌드한다.
