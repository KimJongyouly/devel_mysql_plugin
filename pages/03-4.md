# 03-4. 빌드와 등록

### macOS 빌드 및 등록

```bash
cd 03_Median/Mac
make
sudo make install
```

```sql
-- MySQL에 집계 함수로 등록
CREATE AGGREGATE FUNCTION my_median RETURNS REAL SONAME 'my_median.so';
```

스칼라 UDF와 다른 점은 `CREATE AGGREGATE FUNCTION`을 사용한다는 것입니다. `RETURNS REAL`은 `double`을 반환한다는 의미입니다.

### Linux 빌드 및 등록

```bash
cd 03_Median/Linux
make
make install
```

```sql
USE test;
DROP FUNCTION IF EXISTS my_median;
CREATE AGGREGATE FUNCTION my_median RETURNS REAL SONAME 'median.so';
```
