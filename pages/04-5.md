# 04-5. MySQL STATUS 변수 노출

macOS의 `my_resource` 플러그인은 수집한 메트릭을 `SHOW STATUS`로 조회할 수 있도록 STATUS 변수로도 노출합니다.

### STATUS 변수 등록

```cpp
static double   g_cpu_pct      = 0.0;
static long long g_mem_used_mb = 0;
static long long g_mem_total_mb = 0;

static SHOW_VAR my_resource_status_vars[] = {
    {"my_resource_cpu_pct",
     (char*)&g_cpu_pct, SHOW_DOUBLE, SHOW_SCOPE_GLOBAL},
    {"my_resource_mem_used_mb",
     (char*)&g_mem_used_mb, SHOW_LONGLONG, SHOW_SCOPE_GLOBAL},
    {"my_resource_mem_total_mb",
     (char*)&g_mem_total_mb, SHOW_LONGLONG, SHOW_SCOPE_GLOBAL},
    {NullS, NullS, SHOW_LONG, SHOW_SCOPE_GLOBAL}  // 종료 마커
};
```

이 배열을 플러그인 descriptor에 등록합니다.

```cpp
mysql_declare_plugin(my_resource)
{
    MYSQL_DAEMON_PLUGIN,
    &my_resource_info,
    "my_resource",
    "youly",
    "MySQL OS Resource Monitor",
    PLUGIN_LICENSE_GPL,
    my_resource_init,
    nullptr,
    my_resource_deinit,
    0x0101,
    my_resource_status_vars,  // STATUS 변수 등록
    my_resource_sys_vars,     // 시스템 변수 등록
    nullptr,
    0
}
mysql_declare_plugin_end;
```

### 시스템 변수 (SET GLOBAL)

모니터링 주기를 런타임에 변경할 수 있도록 시스템 변수도 등록합니다.

```cpp
static int g_interval = 5;

static MYSQL_SYSVAR_INT(
    interval,
    g_interval,
    PLUGIN_VAR_RQCMDARG,
    "Monitoring interval in seconds",
    NULL, NULL,
    5, 1, 3600, 0
);
```

등록 후:

```sql
-- 현재 상태 확인
SHOW STATUS LIKE 'my_resource%';
-- +------------------------------+----------+
-- | Variable_name                | Value    |
-- +------------------------------+----------+
-- | my_resource_cpu_pct          | 4.200000 |
-- | my_resource_mem_used_mb      | 7821     |
-- | my_resource_mem_total_mb     | 16384    |
-- +------------------------------+----------+

-- 주기를 10초로 변경
SET GLOBAL my_resource_interval = 10;
```
