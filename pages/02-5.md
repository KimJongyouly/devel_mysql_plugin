### 기본 테스트

```sql
-- macOS/MySQL 버전
SELECT helloworld('Alice');
-- +----------------------------------------+
-- | helloworld('Alice')                    |
-- +----------------------------------------+
-- | Welcome Alice, HelloWorld Mysql Plugin |
-- +----------------------------------------+

SELECT helloworld('홍길동');
-- +-------------------------------------------+
-- | helloworld('홍길동')                      |
-- +-------------------------------------------+
-- | Welcome 홍길동, HelloWorld Mysql Plugin   |
-- +-------------------------------------------+
```

한글 등 UTF-8 멀티바이트 문자도 `args->lengths[0]`이 바이트 길이를 정확히 반환하고 `%.*s`로 처리하므로 올바르게 동작합니다.

### 테이블과 함께 사용

```sql
CREATE TABLE users (name VARCHAR(100));
INSERT INTO users VALUES ('Alice'), ('Bob'), ('홍길동');

SELECT name, helloworld(name) AS greeting FROM users;
-- +---------+-------------------------------------------+
-- | name    | greeting                                  |
-- +---------+-------------------------------------------+
-- | Alice   | Welcome Alice, HelloWorld Mysql Plugin    |
-- | Bob     | Welcome Bob, HelloWorld Mysql Plugin      |
-- | 홍길동  | Welcome 홍길동, HelloWorld Mysql Plugin   |
-- +---------+-------------------------------------------+
```

### NULL 처리 확인

```sql
SELECT helloworld(NULL);
-- +------------------+
-- | helloworld(NULL) |
-- +------------------+
-- | NULL             |
-- +------------------+
```

### 오류 케이스

```sql
-- 인자 없음
SELECT helloworld();
-- ERROR 1123: Can't initialize function 'helloworld';
-- helloworld() requires exactly one string argument (name)

-- 인자가 2개
SELECT helloworld('a', 'b');
-- ERROR 1123: Can't initialize function 'helloworld';
-- helloworld() requires exactly one string argument (name)
```
