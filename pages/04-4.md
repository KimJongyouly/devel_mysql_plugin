# 04-4. OS 메트릭 수집

### Linux: sysinfo() 시스템 콜

Linux에서는 `<sys/sysinfo.h>`의 `sysinfo()` 함수로 시스템 자원 정보를 가져옵니다.

```c
struct sysinfo {
    long uptime;            // 부팅 후 경과 시간 (초)
    unsigned long loads[3]; // 1분, 5분, 15분 평균 부하 (65536 기준)
    unsigned long totalram; // 전체 RAM (mem_unit 단위)
    unsigned long freeram;  // 여유 RAM (mem_unit 단위)
    unsigned short procs;   // 실행 중인 프로세스 수
    unsigned long mem_unit; // 메모리 단위 (바이트, 보통 1)
};
```

CPU 부하 계산:

```cpp
double load = info.loads[0] / 65536.0;
```

`loads[0]`은 1분 평균 부하를 65536으로 스케일링한 고정소수점 값입니다. 65536으로 나누면 실제 load average를 얻습니다.

메모리 계산:

```cpp
unsigned long total_ram = (info.totalram * info.mem_unit) / 1024 / 1024;
unsigned long free_ram  = (info.freeram  * info.mem_unit) / 1024 / 1024;
```

### macOS: Mach API와 IOKit

macOS 버전(`my_resource`)은 훨씬 더 많은 메트릭을 수집합니다.

**CPU 사용률** — Mach API `host_processor_info()`:

```cpp
processor_info_array_t cpu_info;
mach_msg_type_number_t num_cpus_info;
natural_t num_cpus;

host_processor_info(mach_host_self(), PROCESSOR_CPU_LOAD_INFO,
                    &num_cpus, &cpu_info, &num_cpus_info);
```

1초 간격으로 두 번 측정해 그 차이로 CPU 사용률을 계산합니다 (idle 시간 제외).

**메모리 사용량** — `vm_statistics64`:

```cpp
vm_statistics64_data_t vmstat;
mach_msg_type_number_t count = HOST_VM_INFO64_COUNT;
host_statistics64(mach_host_self(), HOST_VM_INFO64,
                  (host_info64_t)&vmstat, &count);

uint64_t used_pages = vmstat.active_count + vmstat.wire_count;
uint64_t used_mb = (used_pages * page_size) / 1024 / 1024;
```

**디스크 I/O** — IOKit `IOBlockStorageDriver`:

macOS의 IOKit 프레임워크를 통해 블록 스토리지 드라이버에서 누적 읽기/쓰기 바이트를 가져오고, 이전 측정값과의 차이로 초당 속도를 계산합니다.
