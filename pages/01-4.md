# 01-4. 빌드와 설치

### Makefile 구조

```makefile
CXX      = g++
TARGET   = helloworld_udf.so
SRC      = helloworld_udf.cpp

MYSQL_CONFIG  = /usr/local/mysql/bin/mysql_config
MYSQL_CFLAGS := $(shell $(MYSQL_CONFIG) --cflags)
MYSQL_PLUGINDIR := $(shell $(MYSQL_CONFIG) --plugindir)

CXXFLAGS = $(MYSQL_CFLAGS) -fPIC -O2 -Wall

UNAME := $(shell uname)
ifeq ($(UNAME), Darwin)
    SHARED_FLAG = -dynamiclib
else
    SHARED_FLAG = -shared
endif

all: $(TARGET)

$(TARGET): $(SRC)
	$(CXX) $(SHARED_FLAG) $(CXXFLAGS) -o $@ $<

install: $(TARGET)
	cp $(TARGET) $(MYSQL_PLUGINDIR)/
```

핵심 컴파일 플래그:

- **`-fPIC`** (Position Independent Code): 공유 라이브러리는 메모리의 어느 주소에 로드될지 모릅니다. `-fPIC`는 절대 주소 대신 상대 주소를 사용하는 코드를 생성합니다. 공유 라이브러리에는 **반드시** 필요합니다.
- **`-dynamiclib` (macOS) / `-shared` (Linux)**: 실행 파일이 아닌 동적 라이브러리(`.so` / `.dylib`)를 생성합니다.
- **`$(MYSQL_CONFIG) --cflags`**: `mysql_config`가 MySQL 헤더 경로를 자동으로 제공합니다.

### 빌드 실행

```bash
cd 01_HelloWorld/Mac   # 또는 01_HelloWorld/Linux
make
```

성공하면 `helloworld_udf.so`(macOS: `.dylib`) 파일이 생성됩니다.

### 플러그인 디렉터리에 설치

```bash
sudo make install
```

또는 수동으로:

```bash
# MySQL 플러그인 디렉터리 확인
mysql_config --plugindir
# 예: /usr/local/mysql/lib/plugin

cp helloworld_udf.so /usr/local/mysql/lib/plugin/
```

Linux MariaDB에서는:

```bash
sudo cp udf_hello_world.so /home/mariadb/mariadb_11.4.3/lib/plugin/
sudo chown mariadb:mariadb /home/mariadb/mariadb_11.4.3/lib/plugin/udf_hello_world.so
sudo chmod 755 /home/mariadb/mariadb_11.4.3/lib/plugin/udf_hello_world.so
```

파일을 복사하는 것만으로는 MySQL에 등록되지 않습니다. 다음 단계에서 SQL 명령으로 등록해야 합니다.

### MySQL에 함수 등록

MySQL 클라이언트에 접속 후:

```sql
CREATE FUNCTION helloworld RETURNS STRING SONAME 'helloworld_udf.so';
```

- `RETURNS STRING`: 반환 타입. `STRING`, `INTEGER`, `REAL`, `DECIMAL` 중 선택합니다.
- `SONAME 'helloworld_udf.so'`: 플러그인 디렉터리에 있는 파일 이름입니다.

MySQL은 이 명령을 받으면 플러그인 디렉터리에서 해당 파일을 열고, `helloworld_init`, `helloworld`, `helloworld_deinit` 세 함수가 존재하는지 확인합니다. 모두 있으면 등록에 성공합니다.

등록 정보는 `mysql.func` 테이블에 저장되어 MySQL 재시작 후에도 유지됩니다.

```sql
SELECT * FROM mysql.func WHERE name = 'helloworld';
```
