# 01-6. 1장 정리

이 장에서 배운 내용을 정리합니다.

- UDF는 C/C++로 작성한 공유 라이브러리를 MySQL에 등록해 SQL 함수로 사용하는 확장 메커니즘이다.
- 스칼라 UDF는 `_init()`, 메인 함수, `_deinit()` 세 가지 C 함수로 구성된다.
- C++로 작성할 때는 `extern "C"` 선언이 필수다.
- `-fPIC` 플래그와 `-shared`(`-dynamiclib`) 플래그로 공유 라이브러리를 빌드한다.
- `CREATE FUNCTION ... SONAME '...'` 으로 등록하고, `DROP FUNCTION` 으로 제거한다.

다음 장에서는 인자를 받는 UDF를 작성하며 `UDF_ARGS` 구조체와 동적 메모리 관리를 배웁니다.
