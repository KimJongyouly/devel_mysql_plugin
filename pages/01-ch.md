# 제1장: Hello World UDF — 첫 번째 플러그인 만들기

MySQL UDF(User Defined Function)의 기초를 다지는 챕터입니다.

가장 단순한 `helloworld()` 스칼라 UDF를 처음부터 만들어봅니다. UDF의 종류(스칼라/집계/스토리지엔진)를 비교하고, 개발 환경 설정부터 빌드, 설치, 동작 확인까지 전 과정을 다룹니다.

```sql
CREATE FUNCTION helloworld RETURNS STRING SONAME 'helloworld_udf.so';
SELECT helloworld();
-- Hello World Mysql Plugin-UDF
```

**주요 학습 내용:**
- UDF의 3-function 구조: `_init()` (1회) → 메인 함수 (행마다) → `_deinit()` (1회)
- `extern "C"` 선언이 필요한 이유 (C++ name mangling 방지)
- `-fPIC` / `-shared` 컴파일 플래그
- `CREATE FUNCTION` / `DROP FUNCTION` SQL 구문
- 자주 마주치는 설치 오류와 해결 방법
